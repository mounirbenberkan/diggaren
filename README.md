# Diggaren
Diggaren är ett projekt som gör det möjligt för användare att se vilken låt
som spelas just nu, eller senast spelades, på Sveriges Radio och enkelt
fortsätta lyssna på musiken på Spotify.

## Setting up Enviroment Variables
Before running the application, you need to set up environment variables for your Spotify API credentials.

### On Linux/macOS Terminal:
```bash
export SPOTIFY_CLIENT_ID=your-client-id
export SPOTIFY_CLIENT_SECRET=your-client-secret
```
### On Windows PowerShell:
```powershell
$env:SPOTIFY_CLIENT_ID="your-client-id"
$env:SPOTIFY_CLIENT_SECRET="your-client-secret"
```
